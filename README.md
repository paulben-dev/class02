# 作业批改助手 — 家长端演示版

纯静态网站，模拟微信小程序中的家长端作业批改功能。

## 启动方式

直接用浏览器打开 `index.html` 即可运行，无需任何构建工具或服务器。

```bash
# 最简单的方式
open index.html

# 或者用 HTTP 服务器（如需手机预览）
python3 -m http.server 8080
# 然后访问 http://localhost:8080
```

## 技术栈

- HTML5 + CSS3 + Vanilla JavaScript (ES6+)
- 零外部依赖
- SVG 内联绘制环形图
- localStorage 持久化订阅状态
- @media print 支持 PDF 导出

## 功能模块

| Tab | 功能 | 说明 |
|-----|------|------|
| 📋 作业 | 作业列表 + 打印 | 浏览8份模拟作业，搜索筛选，一键打印导出PDF |
| 📷 批改 | 拍照上传 + AI批改 | 模拟拍照/文件上传，进度动画，自动评分纠错 |
| 📊 报告 | 批改报告 | 单次/每周/每月三种视图，成绩仪表盘、错因分布、知识点三级分层、趋势图、学习建议 |
| 📝 错题本 | 历史错题 | 知识点筛选、7天免费限制、付费订阅解锁 |
| 👤 我的 | 个人中心 | 孩子信息、订阅管理（¥9.9/月） |

## 四个核心流程

1. **作业打印**：作业列表 → 点击[打印] → 浏览器打印对话框 → 导出PDF
2. **拍照批改**：选择作业 → 拍照/上传 → [提交批改] → 进度动画 → 得分+逐题纠错
3. **查看报告**：成绩仪表盘 → 错因分布(%) → 知识点三层分级(掌握/波动/薄弱) → 典型错题分析 → 学习建议
4. **错题复习**：知识点筛选 → 时间筛选 → 7天免费查看 → 订阅解锁全部

## 模拟数据

- 5门科目：数学、语文、英语、物理、化学
- 20份作业（含13份已批改）
- 30+道错题
- 42位同班同学
- 28个知识点

## 演示说明

- **订阅功能**：点击任意"订阅"按钮 → 模拟支付 → 状态持久化到 localStorage
- **打印功能**：点击打印 → 动态生成PDF → 弹出PDF预览 → 可下载PDF或转发微信好友（Web Share API）
- **拍照模拟**：生成彩色占位图片，支持排序和删除
- **7天限制**：需修改 `firstGradingDate` 测试，默认从首次批改日起算

## 浏览器兼容

- Chrome / Edge / Safari / Firefox 最新版
- 移动端浏览器（414px 适配，模拟手机屏幕）
