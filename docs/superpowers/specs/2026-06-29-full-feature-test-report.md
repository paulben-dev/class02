## Bug 清单

### Bug #1: Mock 批改路径结果不可见

- **严重程度**: 中
- **影响范围**: 演示模式下（API 未连接）批改完成后用户看不到批改结果
- **问题**: `showGradingResultMock()` 设置 innerHTML 但未设置 `display = 'block'`
- **对比**: `showAIResult()` (API 路径) 正确设置了 `container.style.display = 'block'`
- **位置**: `index.html` showGradingResultMock 函数
- **现象**: 批改完成后只显示"批改完成！"，详细结果不可见（DOM 中存在但 hidden）
